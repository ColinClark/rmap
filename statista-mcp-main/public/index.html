Overview

This MCP server provides access to Statista's comprehensive statistical database through two powerful tools. Connect with any MCP-compatible client to search and retrieve statistical data.

Purpose: Search Statista's database for relevant statistical content

Quick Start

1. Check Server Health

Verify the server is running:

curl /health

2. Connect Your MCP Client

Use this endpoint in your MCP client configuration:

/mcp

3. Authentication

Include your Statista API key in the Authorization header:

Authorization: Bearer YOUR_STATISTA_API_KEY

Example Workflow

  1. Use search-statistics with a query like "electric vehicle sales"
  2. Review the returned statistics and note the identifier of interest
  3. Use get-chart-data-by-id with that identifier to get detailed data
  4. Analyze the comprehensive chart data, sources, and methodology

Requirements

  • Valid Statista API key
  • MCP-compatible client (Claude Desktop, custom clients, etc.)
  • Network access to Statista's API endpoints
